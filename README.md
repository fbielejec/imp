# Imp

Web-app for calculating and displaying spatial statistics.
Backend implemented in Clojure, frontend in React + D3, deployed on Heroku.
http://tree-imp.herokuapp.com/

## Usage

lein run

Point browser to http://localhost:8080/

### Dev

lein test
